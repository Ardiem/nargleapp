Today, you're going to make this site a lot bigger.

Instructions for getting started

  1. Go to this repository on github.com.
  2. If you have a github account, fork the repository using the button on the upper right. Clone or download the forked repository into a directory of your choosing using `git clone <ssh_link>` on your command line.

    If you do not have a github account, you can either make one and do the same, or you can clone or download this repository without the intent of submitting a pull request.
  3. Once in your folder, type `python application.py` into your command line and navigate to the IP provided in the output. This is your local development IP and will reflect any changes you make immediately.
  4. Complete one of the tasks assigned below. You can use layout.html to get started. Make sure to make your file and route names unique so that you don't collide with anyone else!
  5. Add a link to your new page to home.html between the <ul> tabs.
  6. Commit your changes using `git commit`.
  7. To begin creating a pull request, push your changes to your fork using `git push -u origin HEAD`.
  8. Return to your forked repository, and select to create a pull request. Set the pull request to compare to the original repository.
  9. If there's time, I'll accept your PRs and show you how I would deploy these changes.

Tasks

  1. Includes a gif
  2. Links to a different site
  3. Has text in three different columns
  4. Has a block of text with a border
  5. Has a hot pink background
  6. Includes a code block
  7. Has four different headings in different styles
  8. Includes a footnote and an anchor tag
  9. Includes two images side-by-side with some space between them
  10. Includes an ordered list and an unordered list
